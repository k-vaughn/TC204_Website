# Ontology Registry

This page lists all known ontologies included in the RITSO.

|Prefix | Official IRI | RITSO Location | Description |
|-------|--------------|----------------|-------------|
| itsCore | https://w3id.org/itsdata/core/ | ontology-its-core | This ontology defines core concepts related to intelligent transport systems (ITS). |
| itsRegulation | https://w3id.org/itsdata/regulation/ | ontology-its-regulation | This ontology defines concepts related to traffic regulations. |
| itsVocabulary | https://w3id.org/itsvocab/ | iso14812 | concrete or abstract thing that exists, did exist, or can possibly exist, including associations among these things |
